## SVG Profile Icon component

A basic SVG that shows a profile/avatar shape in the currentColor by default, or

### Default ProfileIcon

```js live
<ProfileIcon />
```

### Custom color by using the fillColor prop

Modify the text to see it change live:

```js live
<ProfileIcon fillColor="teal" />
```
