## SVG Lock Icon component

A basic SVG that shows a padlock shape in the currentColor by default

### Default ProfileIcon

```js live
<LockIcon />
```

### Custom color by using the fillColor prop

Modify the text to see it change live:

```js live
<LockIcon fillColor="purple" />
```
